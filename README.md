# Sistema de Banco de Dados - Livraria
Projeto de modelagem e implementação de banco de dados para sistema de livraria, utilizando MySQL Workbench. Inclui DER, normalização, criação de tabelas, chaves primárias e estrangeiras, além de controle de vendas e movimentação de estoque.

## Entidades
- Usuario
- Cliente
- Livro
- Venda
- Item_Venda
- Mov_Estoque

## Funcionalidades
- Controle de vendas
- Controle de estoque
- Registro de movimentações

## Tecnologias
- MySQL
- MySQL Workbench
